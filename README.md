# Noleggio Auto - Database Project

Progetto sviluppato per il corso di **Basi di Dati** del Corso di Laurea Triennale in Informatica (Università degli Studi di Padova).

Il progetto consiste nella progettazione e implementazione di una base di dati PostgreSQL per la gestione di un'azienda di noleggio veicoli. Comprende:

- Analisi dei requisiti
- Progettazione concettuale (schema E-R)
- Progettazione logica
- Implementazione dello schema relazionale in PostgreSQL
- Popolamento del database con dati di esempio
- Query SQL di interrogazione
- Ottimizzazione tramite indici
- Applicazione client in C che interagisce con il database tramite libpq

## Tecnologie

- PostgreSQL
- Linguaggio C
- libpq
- SQL
